Resources / Learning

Programming

  • Atlassian guide to Git
  • Computational Methods in the Civic Sphere — aka "Generally how to make your computer do things for you, with a particular focus on data journalism sorts of things." Their Bash guide is super approachable.
  • Full Stack Python — Python IRL. Good conceptual overviews and detailed explanations of how to use Python to write web apps.
  • Hitchhiker's Guide to Python — aka the Python bible.
  • Sparkfun has a lot of very readable tutorials for EE concepts.

Being outside

  • Bay Area Hikes
  • Bay Area Anything Active meetup group. I haven't actually been to any of these (yet) because I don't have a car, but the event archive has a lot of hiking ideas
  • Highline Kites — Small kite shop at Chavez Park next to the Berkeley Marina. They host the super awesome annual kite festival.

Random STEM websites

  • The original Sojourner Rover website from 1996-1997 is still up and is one of my favorite things on the internet. Some of the subpages have text and HIGH GRAPHICS VERSIONS. It actually used a lot of commodity hardware, and I was surprised to learn the controls had a VR component.
  • Berkeley seminar CS39 — fun intro to symmetry and topology that I took way back in my sophomore year. I guess the site itself isn't super great, but clicking around can set you off down rabbit holes of knots and symmetry and graphs.

Things I like reading

  • Wait But Why
  • Scientific American Q&A with Scott Aaronson
  • Actually, the blog blog which produced that^ actually has consistently interesting posts
  • Priceonomics — Blog posts on a variety of topics, usually based on datasets. These seem to be written by companies trying to increase their SEO, but the article quality is usually pretty good. Often covers SF-related topics.
  • The Halting Problem — The Onion, for tech. (Written by Peter Gao.) I can't tell if it's hard to tell these are satire because they're so excellently written, or because Silicon Valley really has lost its mind. Probably both.
  • The Martian — If you haven't read The Martian you need to read The Martian.
