Analyzing Kenya's electrical grid

Summer 2013 - Technology and Infrastructure for Emerging Regions Lab (TIER) - UC Berkeley

I worked as a research assistant to Javier Rosa and analyzed tweets from @KenyaPower, the customer service account of the Kenya Power and Lighting Company, using the Twitter API and a Naive Bayes classifier to see if power outage patterns could be predicted from complaints. Rudimentary visualization was implemented using Google Fusion Tables and Google Maps APIs.

Fencing Website

Completely remade the UC Berkeley Fencing Club's website for the Fall 2014 semester. First time doing any sort of web dev (better late than never), so I decided that using Twitter Bootstrap and Wordpress would be the best way to quickly make a modern-looking and responsive site. The old website used a custom template from an unknown source held together with some spaghetti PHP, cutenews for annoucnements, and jalbum for photos. Although wordpress is a lot heavier than cutenews, it seems to be a lot better supported, and the admin interface more accessible to non-technical people. Likewise I changed the photo hosting to Flickr, which allows anyone on the team to easily upload photos, and slideshows can be easily embedded. I also uploaded the teaching videos to YouTube and converted the shorter ones into gifs so that they are more accessible.
